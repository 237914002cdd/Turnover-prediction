**文档状态**: v1.0 · 2026-06-12  
**适用范围**: 销售定价、客户报价、投资人财务模型  
**定价模型**: 双轨制——云端 SaaS 订阅 + 私有部署算力授权  
**目标客群**: 中大型企业（1,000–50,000 人）、金融/医疗/科技行业

## 1. 定价哲学

传统 HR Tech 定价通常按 **员工人头（Per-Seat）** 收费——这意味着企业规模越大，费用线性增长，但产品价值并非线性增长。

Turnover ONA 采用 **价值导向定价（Value-Based Pricing）**：
- **核心假设**: 每个高风险离职事件的平均成本为 ¥20,000–¥50,000
- **定价锚点**: 我们的定价应 ≤ 客户因一个关键人才离职所产生的替换成本的 **10%**
- **定价目标**: 让客户在"买一个工具"和"冒一次关键离职"之间做选择题——答案不言自明

### 1.1 定价原则

| 原则 | 说明 |
|------|------|
| **价值对齐** | 价格与客户从 ONA 洞察中获得的财务回报挂钩，而非员工数量 |
| **低门槛进入** | 提供 Lite 版本让客户 15 分钟内完成数据导入→获取第一份报告，降低决策成本 |
| **按年锁价** | 首年合同锁定价格，次年根据数据量级和功能深度梯度调整 |
| **合规溢价** | 金融/医疗行业的私有部署版本含合规审计模块，价格上浮 60–100% |

## 2. 定价矩阵

| **Professional** | 中型企业（1,000–5,000 人） | ≤ 5,000 | Lite + SHAP 归因面板、ROI 沙盘、多重干预、Playbook | ¥149,800 |
| **Enterprise** | 大型企业（5,000–50,000 人） | ≤ 50,000 | Professional + 全局 Cascade 分析、SSO/RBAC 认证、自定义数据源接入 | ¥499,800 |

**附加说明:**
- 超出节点上限部分按 ¥1/节点/年计费

- 企业版含专属客户成功经理（CSM） + 季度业务复盘

### 2.2 私有部署算力授权买断（金融/医疗/政务行业）

| 授权类型 | 适用场景 | 授权费（一次性） | 年维保费 | 说明 |
|---------|---------|---------------|---------|------|
| **Standard On-Prem** | 一般私有化需求 | ¥698,000 | 授权费的 15% | 含 ONA 引擎 + SHAP + 沙盘，不含数据源 ETL 适配 |
| **Compliance-Grade** | 金融/医疗行业 | ¥1,198,000 | 授权费的 15% | Standard + 审计日志、数据加密、部署架构审查报告 |
| **Enterprise Suite** | 政务/军工 | 按需报价 | 协商 | Compliance-Grade + 国产化信创适配（ARM 架构、达梦/人大金仓数据库） |

**私有部署价格锚点逻辑:**
- 金融行业单人替换成本约 ¥35,000–¥60,000
- 以一个 5,000 人规模的金融客户计算：Compliance-Grade ¥1.198M ÷ 5,000 = ¥240/人
- 这低于一杯精品咖啡的价格——而它覆盖了整个 ONA 风险识别与干预闭环

## 4. 销售流程与定价组合

### 4.1 漏斗模型

```
发现阶段        免费 Demo + 样本数据 + CHO 演示脚本 → 0 成本
    ↓
POC 阶段        Lite 版 30 天试用 + 客户真实数据导入 → ¥0（人力成本投入）
    ↓
签约阶段        Professional / Enterprise 年度订阅 → 首年 ¥149,800–¥499,800
    ↓
扩展阶段        私有部署 / 额外数据源 / 模型调优 → ¥698,000+
    ↓
续约阶段        次年续费含增值服务 → 年维保费 ¥74,970+
```

## 5. 附加条款

### 5.1 数据隐私和服务等级（SLA）

| 版本 | 数据存储 | 数据加密 | 可用性 | 支持响应 |
|------|---------|---------|--------|---------|
| Lite / Professional | 云端多租户（阿里云/腾讯云） | TLS 1.3 + AES-256 | 99.5% | 8h 工单 |
| Enterprise SaaS | 云端独立实例 | TLS 1.3 + AES-256 + KM | 99.9% | 2h 工单 + CSM |
| On-Prem 私有部署 | 客户自管服务器 | 可配置加密策略 | N/A（客户自管） | 4h 工单 + 专属 TAM |

### 5.2 退款与终止

- 首月内无条件退款（扣除实施成本）
- 连续 2 个月可用性低于 SLA → 按比例退款
- 数据导出窗口：合同终止后 90 天内提供全量数据导出
